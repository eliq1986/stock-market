# Stock Market

Stock/News app that pulls two APIs and displays stock data and world news. Current build includes only mobile and not desktop/tablet.

Built utilizing mobile first approach.

## Technologies
	- JavaScript
		- React
	- CSS3
		- FlexBox
	- HTML5
	- NPM
		- Create-React-App
